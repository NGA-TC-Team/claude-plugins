# NextGenAI AX 컨설팅 플러그인

NextGenAI의 AX(AI Transformation) 컨설팅 방법론을 표준화한 플러그인입니다. 진단→설계→빌드→확산 4단계 프레임워크에 따라 누가 작업하더라도 일관된 품질의 컨설팅 산출물을 생성할 수 있습니다.

## 구성 요소

### Skill: AX 방법론 (ax-methodology)

NextGenAI의 전체 AX 컨설팅 방법론 지식을 담고 있습니다.

- 4단계 프레임워크: 세일즈 → 진단 → 빌드 → 확산
- 8단계 통합 자동화 구축 프로세스
- 역할 정의 (세일즈팀/진단팀/빌드팀)
- 5대 원칙 및 PM 가이드
- AI 리터러시 레벨링 기준
- 자동화 후보 선정 방법론

AX 관련 질문이나 컨설팅 과정에서 자동으로 참조됩니다.

### Commands

| 명령어 | 설명 |
|--------|------|
| `/진단보고서 [고객사명]` | 프로세스 진단 결과를 PPT 보고서로 생성 |
| `/전략제안서 [고객사명]` | AI 전환 전략 및 로드맵 제안서 PPT 생성 |
| `/실행계획서 [고객사명]` | 주차별 실행 계획서 PPT 생성 |
| `/경영진보고 [고객사명]` | 주간/월간/완료 보고 PPT 생성 |

각 커맨드는 필요한 정보를 질문한 뒤, NextGenAI 방법론에 맞는 구조화된 PPT를 생성합니다.

## 사용 방법

1. 플러그인 설치 후, 슬래시(/) 커맨드로 산출물을 생성합니다.
2. 고객사 정보를 입력하면 방법론에 따라 일관된 구조의 PPT가 만들어집니다.
3. AX 관련 질문을 하면 방법론 스킬이 자동으로 참조되어 답변합니다.

## 설정

별도의 환경 변수나 외부 서비스 연동은 필요하지 않습니다.
